# Mutual Fund Analytics Platform

## Bluestock Fintech Data Analyst Internship Capstone Project

### Project Overview

This project analyzes mutual fund performance, investor behavior, portfolio diversification, and market trends using Python, SQL, and Power BI.

The objective is to build a complete analytics pipeline that transforms raw mutual fund data into actionable insights through data engineering, exploratory analysis, financial metrics, and interactive dashboards.

---

## Project Objectives

* Build ETL pipelines for financial datasets
* Clean and validate mutual fund data
* Design an analytics database using SQLite
* Perform exploratory data analysis
* Calculate risk and performance metrics
* Develop advanced financial analytics
* Create an interactive Power BI dashboard
* Generate investment insights and recommendations
